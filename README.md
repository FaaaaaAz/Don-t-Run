# Don't Run

## Descripción

**Don't Run** es un juego desarrollado en Unity donde el jugador debe explorar un laberinto para encontrar una llave y escapar.

El objetivo principal es sobrevivir y encontrar la salida lo más rápido posible.

---

## Mecánicas principales

* Movimiento del jugador en entorno 3D
* Recolección de objetos (llave)
* Sistema de interacción (teclas E y F)
* Apertura de puerta para completar el nivel

---

## Controles

* WASD → Movimiento
* Mouse → Cámara
* E → Recoger objetos
* F → Interactuar 

---

## Tecnologías utilizadas

* Unity (Motor de juego)
* C# (Programación)

---

## Objetivo del juego

Encontrar la llave y abrir la puerta para escapar del laberinto.
